# Hover Photo Project

## Overview
The Hover Photo Project implements a hover effect that displays a photo following the mouse cursor when hovering over specified target elements. This project is built using JavaScript and CSS, providing a smooth and interactive user experience.

## Files Structure
- **src/**
  - **hover-follow.js**: Contains the `HoverPhoto` class that manages the hover effect.
  - **main.js**: The entry point of the application that initializes the hover effect.
  - **styles/**
    - **hover.css**: CSS styles for the hover effect, including animations and transitions.
- **public/**
  - **index.html**: The main HTML file that includes the necessary scripts and styles.
- **package.json**: Configuration file for npm, listing dependencies and scripts.
- **.gitignore**: Specifies files and directories to be ignored by Git.
- **README.md**: Documentation for the project.

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd hover-photo-project
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage
1. Open `public/index.html` in a web browser.
2. Hover over the designated target elements to see the photo follow your cursor.

## Customization
- You can modify the photo displayed by changing the image source in the `hover-follow.js` file.
- Adjust the styles in `src/styles/hover.css` to customize the appearance of the hover effect.
